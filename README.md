# Découvrons les sons! (Jeu Unity)
Un jeu éducatif sur les syllables et les phonèmes

## Comment jouer?
Pour controller chaque personnage, utilisez les touches WASD et les flèches\
Selectionnez le bon choix avec Shift\
Pour recevoir des points bonus, clickez sur Shift lorsque la jauge est sur la zone bleue!\
